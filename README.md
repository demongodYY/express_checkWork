利用了Nodejs+EXPRESS＋MongoDB完成的公司考勤系统

###功能
- 每日员工考勤状况登记
- 查看特定日期员工考勤状况
- 员工请假
- 查看员工状态
- 添加员工



###需求
- NodeJs
- NPM
- MongoDB


### 安装运行

```
git clone https://github.com/demongodYY/express_checkWork.git
cd express_checkWork
npm install
npm start

```

###使用方法

使用WEB方式访问目标机3000端口
